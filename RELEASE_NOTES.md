- ✅ **1,715+ 精选技能** - 涵盖 28 个核心领域，从 Web 开发到 AI/LLMs
- ✅ **响应式网站** - 完美支持桌面、平板和移动设备
- ✅ **实时搜索** - 快速搜索和多条件筛选功能

- ✅ **社交分享** - Open Graph 和 Twitter Card 支持

## 🛠️ 技术栈

- **前端框架** - React 19 + TypeScript
- **路由** - Wouter
- **样式** - Tailwind CSS 4
- **UI 组件** - shadcn/ui
- **构建工具** - Vite
- **部署** - GitHub Pages
- **SEO** - 元标签、结构化数据、sitemap

## 🚀 开始使用

### 1. 访问网站
👉 [https://dfds2989-source.github.io/awesome-openclaw-skills/](https://dfds2989-source.github.io/awesome-openclaw-skills/)

### 2. 浏览技能
- 在首页查看热门技能
- 使用搜索功能查找特定技能
- 按分类浏览技能
- 查看技能详情和使用示例

### 3. 下载资源
- 点击浮动下载按钮或首页 CTA
- 访问 UC 网盘下载完整的 DOCX 文档
- 获取详细的使用说明和安装命令

### 4. 本地开发
```bash
git clone https://github.com/dfds2989-source/awesome-openclaw-skills.git
cd awesome-openclaw-skills
pnpm install
pnpm run dev
```
